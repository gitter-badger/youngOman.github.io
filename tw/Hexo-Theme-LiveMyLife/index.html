什麽是 Hexo?

Hexo 是一個快速、簡潔且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在幾秒內，即可利用靚麗的主題生成靜態網頁。

評論系統設置

Hexo-Theme-LiveMyLife 主題暫時支持三種評論方式。我使用 gitalk 為默認評論系統。

Gitalk

Gitalk 是基於 GitHub Issue 和 Preact 的現代化的評論組件。 有關詳細的配置方法，請參考 Gitalk 官方文檔。

Gitment

Gitment 是一個基於 GitHub Issues 的評論系統，可以在前端使用它，而無需任何服務器端實現。有關詳細的配置方法，請參見 Gitment 官方文檔。

Disqus

如果你想要使用 Disqus 評論系統，則必須有代理。

Anchorjs（錨點）設置

如果你想要更改文章目錄標題前部的錨點 “❡”，則可以到 layout/_partial/anchorjs.ejs 目錄下進行更改。如何使用 anchorjs，請參閱 AnchorJS 以獲取詳細示例。

博文置頂

Hexo-theme-livemylife 主題添加了文章頂部功能，只要在您的 markdown 筆記中添加屬性sticky: number 配置，文章按此數字排序。
